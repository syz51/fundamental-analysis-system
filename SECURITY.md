# Security Policy

## Reporting Issues

This is an educational/portfolio project demonstrating multi-agent systems architecture. It is not intended for production use or handling sensitive data.

Report any security concerns or vulnerabilities via [GitHub Issues](../../issues).

## Scope

- Educational project only
- Not for production use
- Should not be used with real API keys, credentials, or sensitive financial data
- See disclaimer in README

## Dependencies

Security updates for dependencies are monitored via GitHub Dependabot.
